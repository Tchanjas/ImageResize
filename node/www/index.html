The service that you need to resize your images

A simple high performance web based system running Node.js and NGINX to resize images.

We use several approaches to improve to have a high performance, like load balancing and caching.

Using sharp, a high performance Node.js image processing library, the fastest module to resize JPEG, PNG, WebP and TIFF images. Uses the libvips library.
